# VIDM-v1-
A video media player like VLC still working on it


Vidm is a 100% customizable, high-performance, open-source video and media player built for power users. Engineered to be lightweight yet infinitely modular, OmniPlayer gives you complete control over your playback environment, UI/UX aesthetics, and asset pipelines.

Think of it like VLC, but stripped of legacy bloat, written with modern optimization frameworks, and entirely built to be tailored by you.

## ✨ Features

* **Complete UI/UX Customization:** Modify everything from custom themes, layouts, and window-blur aesthetics (glassmorphism) using an external configuration layer.
* **Hardware Accelerated Decoding:** Seamless GPU-bound rendering pipelines to handle 4K/8K playback effortlessly without straining CPU threads.
* **Infinite Plugin Extensibility:** Inject custom shaders, real-time video filter layers, audio visualizers, or custom streaming protocols via a modular plugin API.
* **Broad Codec Ecosystem:** Native architecture designed to interface flawlessly with modern multimedia frameworks (`FFmpeg`, `GStreamer`, `VA-API`).
* **Cross-Platform & Lightweight:** Zero telemetry, lightning-fast boot times, and a minimalist footprint.

## 🛠️ Tech Stack & Dependencies

* **Core Runtime:** C / C++ (Optimized memory allocation and thread safety)
* **Multimedia Engine:** FFmpeg / LibVLC Core Integration
* **Rendering Pipeline:** OpenGL / Vulkan (Hardware acceleration backend)
